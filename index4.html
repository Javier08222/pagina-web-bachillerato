Objetivos de una página web

1Objetivo General

Diseñar y crear páginas web que incluyen diferentes elementos tales como fondo, textos en distintos formatos, alineación y colores, tablas, imágenes, menú de navegación, entre otros.

2Objetivo Específico I

Explorar y utilizar con interés las herramientas de HTML y CSS3 para el desarrollo de habilidades y dominio de este lenguaje de programación.

3Objetivo Específico II

Identificar las propiedades de la tecnología de HTML para colocar estilos a una página web.
