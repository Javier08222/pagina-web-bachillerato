Propósito y lenguaje de la tecnología HTML

Tecnología HTML

La publicación en la World Wide Web se implementa en forma de sitios web. La estructura de un sitio web se asemeja a una revista que contiene información sobre un tema o problema. Así como la revista consta de páginas impresas, un sitio web también consta de páginas web de computadora. muchas herramientas, incluso gratuitas, que te permiten crear paginas web sin saber realmente HTML.

Por lo general, un sitio tiene una página de portada (con una tabla de contenido), en la que hay hipervínculos a sus secciones principales (páginas web). Los hipervínculos también están disponibles en otras páginas web del sitio, lo que permite al visitante moverse libremente por el sitio.

Un sitio web consta de páginas web con hipervínculos. Las páginas web pueden ser interactivas y pueden contener objetos multimedia y dinámicos (cambiando la página según el algoritmo).

Las principales ventajas de los documentos html son

  • La estructura del texto es amena.
  • Es pequeño el volumen de información.
  • No se requiere de grandes conocimientos, mientras se tenga un editor de web.
  • Permite todos los exploradores.

    • Para crear páginas web, se utilizan editores de texto simples que no incluyen caracteres de control de formato de texto en el documento creado. Como tal editor en Windows, puede usar la aplicación estándar de Bloc de notas

    Conceptos básicas de HTML

  • Un descriptor (etiqueta) es el principal elemento de codificación adoptado en el estándar HTML. Casi todo depende de los descriptores, sirven para llamar la atención sobre ciertas palabras en el documento, dejando en claro al navegador lo que debe resaltarse, dónde mostrar los gráficos y dónde se encuentran los hipervínculos. La etiqueta está entre corchetes angulares. Las etiquetas pueden ser individuales o emparejadas. Para emparejado, es necesario tener una etiqueta de apertura (la mayoría de las veces con un parámetro y su valor) y una etiqueta de cierre.
  • Un contenedor es un par de descriptores que se colocan en la apertura y cierre (etiquetas). Los contenedores están diseñados para almacenar cierta información, como texto u otros descriptores HTML.
  • Un elemento de un documento HTML es un contenedor con su contenido.

    • Todo el contenido del archivo de la página web está encerrado en un contenedor, indicando al navegador que el texto es documento HTML y posiblemente contiene etiquetas que el navegador debe identificar, reconocer e interpretar.
